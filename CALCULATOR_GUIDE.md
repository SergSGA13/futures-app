# 📊 Калькулятор сигналов - Руководство

## Что это?
Калькулятор сигналов - это мощный инструмент анализа торговельных сигналов с динамической фильтрацией по периодам, дням недели и часам работы.

## Как использовать?

### 1. Выберите период
- **Last 30 Days** - анализ за последние 30 дней
- **ALL Periods** - анализ всех данных с начала

### 2. Активные дни недели
По умолчанию выбраны все дни (Пн-Вс). Вы можете деактивировать дни, когда не торгуете:
- Снимите галку с дня, чтобы исключить его из расчетов
- Данные пересчитаются с учетом только выбранных дней

### 3. Активные часы (0-23)
Выберите часы, когда вы активно торгуете:
- Используйте **"Выбрать все"** для быстрого выбора всех часов
- Используйте **"Очистить"** для удаления выбора
- Минимальное 1 часов для анализа

### 4. Ставка для расчета дохода
Выберите размер ставки, которая будет использована для расчета потенциального дохода:
- ETH: 5-125 USDT
- BTC: 5-250 USDT

### 5. Нажмите "Применить фильтры"
Калькулятор пересчитает все метрики с учетом ваших фильтров

## Что показывает калькулятор?

### Параметры системы
Статичные параметры торговли:
- Max позиций: 5 одновременно
- Ставки: ETH 5-125 USDT, BTC 5-250 USDT
- Экспирация: 10 минут

### Производительность
Динамические метрики на основе фильтров:
- **Total сигналов** - количество сигналов в выбранном периоде
- **WIN** - количество выигрышных сигналов
- **LOSE** - количество проигрышных сигналов
- **WR%** - процент выигрышных сигналов (Win Rate)
- **Месячный доход** - расчет прибыли на основе выбранной ставки
  - Формула: WIN × ставка - LOSE × 125
- **Uptime (дней)** - процент прибыльных дней из активных

### Анализ WinRate (Break-even)
Критические показатели для прибыльности:
- **Break-even WR** - минимальный WR% для безубытка на выбранной ставке
  - Формула: 125 / (ставка + 125) × 100
- **Текущий WR** - ваш фактический WR%
- **Статус** - рекомендация:
  - ✅ **Прибыльно** - текущий WR ≥ break-even
  - ⚠️ **На грани** - текущий WR близко к break-even
  - ❌ **Убыточно** - текущий WR < break-even

### Диаграмма дохода
График кумулятивного прибыли по дням с учетом всех фильтров

### Таблицы анализа
- **По парам** - статистика ETH vs BTC
- **По таймзонам** - анализ по минутам часа (0-14, 15-29, 30-44, 45-59)
- **По часам** - детализация по каждому часу (0-23)

Каждая таблица показывает:
- Total - количество сигналов
- Wins - выигрыши
- WR% - процент выигрышей

## Примеры использования

### Пример 1: Анализ лучших часов
1. Период: Last 30 Days
2. Дни: все дни
3. Часы: 8, 9, 10, 11 (утренние часы)
4. Ставка: 100 USDT
5. **Результат:** увидите WR% и доход именно в эти часы

### Пример 2: Выходные не торгуются?
1. Дни: Пн-Пт (снимите галку с Сб, Вс)
2. Часы: выберите все
3. **Результат:** расчет будет только по рабочим дням

### Пример 3: Оптимизация ставки
Попробуйте разные ставки (5, 10, 25, 50, 100 USDT) и смотрите:
- Как меняется break-even WR%
- Какая ставка наиболее прибыльна на ваших фильтрах

## Советы

✅ **Регулярно проверяйте**
- Изменился ли WR после смены часов/дней?
- Какие часы наиболее прибыльны?

✅ **Используйте для планирования**
- Если WR% < break-even → нужна коррекция стратегии
- Если есть лучшие/худшие часы → оптимизируйте график

✅ **Экспериментируйте**
- Пробуйте разные комбинации фильтров
- Находите идеальное окно торговли для вашей системы

## FAQ

**Q: Почему цифры отличаются от Google Sheets?**
A: Калькулятор фильтрует данные по вашим критериям. Google Sheets показывает все данные.

**Q: Можно ли сохранить результаты?**
A: Пока нет, но вы можете сделать скриншот или записать важные метрики.

**Q: Как часто обновляются данные?**
A: При каждом нажатии "Применить фильтры" калькулятор заново загружает данные из Google Sheets.

**Q: Что означает uptime?**
A: Процент дней, когда вы закрыли день в плюс. Например, 70% uptime = 7 дней из 10 были прибыльны.
